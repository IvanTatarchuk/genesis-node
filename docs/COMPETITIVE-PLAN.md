# Competitive plan — Genesis Node vs alternatives

## Goal

Position Genesis Node as the preferred AI agent marketplace: win on clarity, trust, conversion, and SEO vs Zapier, Make, Crew AI, Relevance AI, e2b, and similar platforms.

---

## 1. Differentiation (done)

- **/compare page** — “Why Genesis Node” / “Built to beat the rest”:
  - Pay per task (no per-seat subscriptions)
  - Run by voice + navigate by voice
  - Automatic refund if task fails
  - No lock-in, cancel anytime
  - Developers earn 70–90%
  - First result in 60 seconds, demo without signup
- **Homepage** — “Why choose Genesis Node” strip with link to Compare.
- **FAQ** — New question: “How does Genesis Node compare to Zapier, Make…?” with link to Compare.
- **Pricing** — Copy: “No lock-in: cancel anytime.”

---

## 2. Discovery & SEO (done)

- Sitemap: added `/compare`.
- Meta: title and description for Compare page (keywords: “AI agent marketplace alternative”, “Compare”, “Zapier”, “Make”).
- Nav (home + app): link “Compare” / “Why us”.
- Voice: commands “compare”, “why us”, “alternatives” → `/compare`.

---

## 3. Trust & conversion (already in product)

- Refund guarantee (credits if task fails).
- Stripe, “your data stays yours” in footer.
- 50 free credits, no card to start.
- Live demo in 60 seconds without signup.

---

Differentiation is clear on **/compare** and homepage. SEO and navigation (including voice) point to it. Trust and conversion are reinforced in pricing and FAQ. Optional next steps focus on proof, content, and retention.
