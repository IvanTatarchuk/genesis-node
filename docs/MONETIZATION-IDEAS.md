# Перевірка роботи та ідеї для заробітку

## ✅ Перевірка веб-сторінки

- **Збірка:** `npm run build` — успішно (Next.js 16, Turbopack).
- **Маршрути:** 62 сторінки згенеровані (статичні + динамічні), proxy (колишній middleware) працює.
- **Що перевірити вручну:** запустити `npm run dev`, відкрити головні сторінки: `/`, `/login`, `/marketplace`, `/pricing`, `/dashboard`, `/voice`, `/compare`, `/matadora`. Переконатися, що форма логіну, купівля кредитів, буст агента та реферали відкриваються без помилок.

## Що ще можна створити для заробітку

### Швидкі (менше коду)

1. **Преміум-слот у маркетплейсі**  
   Окремий блок «Featured» / «Editors’ pick» — плата кредитами або фікс за тиждень/місяць (додатково до поточного boost).

2. **Донати розробникам**  
   Кнопка «Support creator» на сторінці агента: користувач переказує кредити або фіксовану суму розробнику (платформа забирає %).

3. **Платні шаблони пайплайнів**  
   Маркетплейс шаблонів: «Купити шаблон за X кредитів» — автор отримує частину.

4. **B2B-тариф (Team / Enterprise)**  
   Фіксована щомісячна плата: більше кредитів, workspace, пріоритетна підтримка, SLA. Окрема сторінка `/pricing#team` або `/enterprise`.

### Середні (API + доступ)

1. **API-доступ за підпискою**  
   Тариф «API Pro»: вищі ліміти, окремий ключ, webhooks. Дохід — щомісячна підписка + можливо overage за виклики.

2. **Приватні агенти**  
   «Приватний агент» — доступ лише за посиланням або за оплату кредитами (разово або підписка). Розробник виставляє ціну доступу.

3. **Консультації / onboarding**  
   Бронювання слоту з експертом або support за кредити (наприклад, 50 кр за 15 хв). Кредити списуються, частина йде консультанту.

### Довгострокові

1. **White-label / embed**  
   Компанії вставляють маркетплейс або окремих агентів під своїм брендом; платформа бере щомісячну плату або % від обсягу.

2. **Преміум-розробники**  
   «Verified» / «Pro» developer: вищий % виручки (наприклад, 85–90%), бейдж, пріоритет у пошуку. Платна щомісячна підписка для девів.

3. **Маркетплейс інтеграцій**  
    Платні інтеграції (Slack, Notion, Airtable тощо): одноразова оплата або підписка; платформа і автор ділять дохід.

## Рекомендований порядок

1. Перевірити в браузері головні флоу (логін, кредити, boost, реферали).  
2. Додати **Featured-слот** або **донати розробникам** — швидко і підсилює монетизацію.  
3. Оформити **B2B-тариф** на папері (ціни, ліміти) і додати CTA на `/pricing`.  
4. Далі — API-тариф або приватні агенти, залежно від запитів користувачів.
