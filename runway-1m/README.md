# Runway to $1M

Calculator: when will you hit $1M ARR at your current MRR and monthly growth? Share your runway via URL.

## Run

- `npm install`
- `npm run dev` — open http://localhost:5173
- `npm run build` — output in `dist/`

## Deploy (Vercel)

From this folder: `npx vercel --prod`

Or connect the repo and set root to `runway-1m`. Optional env: `VITE_LEAD_ENDPOINT` (POST endpoint for email capture).
